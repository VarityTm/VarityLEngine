    🚀 VARITY ENGINE: ИНСТРУКЦИЯ ПО УСТАНОВКЕ v1.0 🚀
============================================================

Добро пожаловать! VarityEngine — это движок, который позволяет 
писать программы и игры на русском языке. Выбирайте любой из 
трех способов установки, который вам удобнее:

------------------------------------------------------------
1️⃣ СПОСОБ: САМЫЙ ПРОСТОЙ (АВТОМАТИЧЕСКИЙ) ✅
------------------------------------------------------------
Подойдет для тех, кто хочет всё и сразу:
1. Найдите в папке или скачайте файл "VarityEngine_Setup.exe".
2. Запустите его и следуйте подсказкам (просто жмите «Далее»).
3. Готово! Теперь файлы .var сами станут красивыми и будут 
   открываться двойным кликом. 🎨

------------------------------------------------------------
2️⃣ СПОСОБ: РУЧНАЯ НАСТРОЙКА (БЕЗ УСТАНОВКИ) ⚙️
------------------------------------------------------------
Если вы не хотите ничего устанавливать в систему:
1. Возьмите файл "main.exe" и положите его в любую удобную 
   папку (например, C:\VarityEngine).
2. Нажмите правой кнопкой на любой ваш файл .var.
3. Выберите "Открыть с помощью" -> "Выбрать другое приложение".
4. Найдите наш "main.exe" и ОБЯЗАТЕЛЬНО поставьте галочку 
   «Всегда использовать это приложение». 🔗

------------------------------------------------------------
3️⃣ СПОСОБ: ДЛЯ РАЗРАБОТЧИКОВ (СБОРКА ИЗ КОДА) 👨‍💻
------------------------------------------------------------
Если вы хотите собрать движок сами из исходного кода:
1. Установите Python и библиотеку PyInstaller: 
   Команда: pip install pyinstaller
2. Положите файлы "main.py" и "icon.ico" в одну папку.
3. Открой терминал в этой папке и введите:
   python -m PyInstaller --onefile --icon="icon.ico" main.py
4. Забирайте свой готовый .exe в папке DIST! 📦
